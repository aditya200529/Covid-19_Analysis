Objectives

Analyze the spread of COVID-19 across countries

Identify top affected countries based on confirmed cases and deaths

Study trends over time using cumulative data

Create interactive and visual dashboards for better understanding

Derive insights to support data-driven decision making

Dataset

Global COVID-19 dataset containing:

Country
Date
Cumulative Confirmed Cases
Cumulative Deaths
(Publicly available dataset sourced from reliable COVID-19 repositories)

Tools & Technologies

Python (Pandas, NumPy)
Data Visualization: Matplotlib / Seaborn / Power BI / Tableau
Jupyter Notebook
Git & GitHub

Key Analysis Performed

Country-wise cumulative case and death analysis
Top 10 countries by confirmed cases
Trend analysis over time
Comparison between confirmed cases and deaths
Dashboard creation for interactive exploration

Visualizations

Line charts for case trends
Bar charts for country-wise comparison
Bubble charts to show severity across regions
Interactive dashboards for insights

Insights

Identified countries most affected by COVID-19
Observed growth patterns during peak pandemic periods
Highlighted relationship between confirmed cases and deaths

Conclusion

This project demonstrates practical skills in data cleaning, analysis, and visualization. It showcases how data analytics can be used to understand large-scale real-world problems like a global pandemic.
